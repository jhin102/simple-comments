Simple Comments 테스트 페이지

⚠️ 주의: 로컬 테스트에서는 API가 작동하지 않습니다. 실제 기능을 테스트하려면 Vercel에 배포해야 합니다.

테스트 1: 기본 댓글 위젯

URL: /?id=test-page-1&w=600&h=500&max=5

테스트 2: 넓은 위젯

URL: /?id=test-page-2&w=800&h=600&max=10

테스트 3: 작은 위젯

URL: /?id=test-page-3&w=400&h=300&max=3

사용법

  1. 각 위젯에서 댓글을 작성해보세요
  2. 좋아요 버튼을 클릭해보세요
  3. 다른 페이지 ID로 댓글이 분리되는지 확인하세요
  4. 페이지네이션이 작동하는지 확인하세요
  5. 댓글 삭제 기능을 테스트해보세요

임베딩 코드 예시

<iframe 
  src="https://your-domain.vercel.app/?id=my-page-1&w=600&h=800&max=20"
  width="600" 
  height="800"
  frameborder="0">
</iframe>

배포 방법

  1. vercel --prod 명령어로 배포
  2. Vercel Postgres 데이터베이스 설정
  3. 환경 변수 설정
  4. 실제 기능 테스트
